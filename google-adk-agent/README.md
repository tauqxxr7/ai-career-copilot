\# 🚀 Google ADK + MCP Agent (Track 2)



This project demonstrates building an AI agent using \*\*Google Agent Development Kit (ADK)\*\* with \*\*Model Context Protocol (MCP)\*\*.



\## 🔧 What I Built

\- AI agent using Google ADK

\- Integrated with:

&#x20; - BigQuery (data insights)

&#x20; - Google Maps (location intelligence)

\- Used MCP to connect tools with the agent



\## 💡 Use Case

The agent helps answer real-world questions like:



"Find the best location to open a bakery in Los Angeles based on foot traffic and competition."



\## 🧠 Features

\- Multi-tool reasoning

\- Real-world data integration

\- Location intelligence

\- Business decision support
